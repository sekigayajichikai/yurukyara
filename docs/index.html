イベントについて

関ヶ谷自治会では、生成AIを活用して自治会の公式ゆるキャラを作るイベントを開催します！

子どもから大人まで、誰でも最先端のAIツールに触れながら、関ヶ谷らしいキャラクターを作っていきます。完成した作品は後日、自治会のみなさんによる投票でグランプリを決定し、選ばれた作品が関ヶ谷自治会の公式キャラクターとして採用されます✨

ものづくりの楽しさと、これからの時代を生きる「AIとの付き合い方」を、地域のみんなで一緒に体験しましょう。
